---
title: Internationalization
description: How Marine Guardian handles interface languages.
---

Marine Guardian's interface is **translatable**: UI labels, buttons, navigation,
and system messages are provided as translation message sets rather than
hard-coded text, so the application can be shown in more than one language.

Data that originates in EarthRanger (event reports, subject names, and other
field-entered content) is displayed **as-is** in its original language — only the
application's own interface text is translated.

- [**Supported Languages**](/docs/i18n/supported-languages) — the languages
  available and where to switch.
