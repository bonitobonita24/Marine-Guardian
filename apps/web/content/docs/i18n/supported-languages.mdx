---
title: Supported Languages
description: The interface languages Marine Guardian ships with.
---

Marine Guardian ships interface translations for three languages:

| Language | Code |
|----------|------|
| English | `en` |
| Bahasa Indonesia | `id` |
| Bahasa Malaysia | `ms` |

English is the default interface language.

## Choosing a language

- A **language switcher** is available so users can change the interface
  language.
- Each user has a **language preference** that is set when their account is
  created and travels with the account.

## What gets translated

The application's own UI text — labels, buttons, navigation, and system messages —
is translated. **Content synced from EarthRanger is not translated**; it appears
in whatever language the field report was written in.
