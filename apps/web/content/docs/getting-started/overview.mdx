Marine Guardian brings a tenant's EarthRanger data together into a set of
purpose-built operations surfaces. Each lives under the tenant path
(`/[tenant]/...`) and is reachable from the left sidebar.

## The core surfaces

- **Command Center (War Room)** — the primary large-display view: live map,
  real-time event feed, and an alert panel. Built for an operations-room screen.
- **Dashboard** — KPIs and a live activity feed in a data-dense, multi-panel
  layout.
- **Live Map** — a full-screen map with layers, heatmaps, and drawing tools.
- **Events** — the operations list and the Kanban board for moving events
  through their states (`new → active → resolved`) and filling in details.
- **Patrols** — monitor active patrols and review completed patrol tracks
  against planned patrol-area polygons.
- **Patrol Areas & Schedule** — draw patrol-area polygons and schedule ranger
  assignments on a Gantt chart.
- **Fuel** — log bulk fuel allocations per area and view consumption analytics.
- **Reports** — per-area, consolidated, detailed event-log, and ranger
  performance reports.
- **Alerts & Notifications** — configure alert rules and review the in-app
  notification center.
